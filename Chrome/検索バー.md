# 検索バー

## メモ帳になる

```
data:text/html, <html contenteditable>
```

## Google ドライブを検索

```
chrome://settings/searchEngines
```

に移動して、「その他の検索エンジン」に

```
Google Drive
```

```
drive
```

```
https://drive.google.com/drive/search?q=%s
```

を設定する

既にある場合は、`docs.google.com` を `d` とか `drive` にすると
検索バーで

```
drive ファイル
```

で検索することができる
