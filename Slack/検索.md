# 検索



### リアクションがないメッセージを検索する例

```
in:#youtube -has:reaction -from:@GitHub (Legacy) -from:@GitHub -is:thread
```

- `in:#youtube` : チャンネル #youtube 内を検索
- `-has:reaction` : リアクションがないメッセージを検索
- `-from:@GitHub (Legacy)` : ユーザー @GitHub (Legacy) 以外のメッセージを検索
- `-from:@GitHub` : ユーザー @GitHub 以外のメッセージを検索
- `-is:thread` : スレッドでないメッセージを検索

### アクションがあるメッセージを検索する例

```
in:#fun has:action -has:reaction
```

- `in:#fun` : チャンネル #fun 内を検索
- `has:action` : アクションがあるメッセージを検索
- `-has:reaction` : リアクションがないメッセージを検索
