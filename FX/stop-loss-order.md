# FX - 逆指値注文

## 常に証拠金維持率が 300% を超えている人

損切りする必要はないです

## それ以外の人

損切りできずにコツコツと負債を貯めてしまう人は損切りする必要があります

ただ、単純に損切りをすればいいのではない

### エントリーしたときの理由を記録して、その理由に反する値動きになったら損切りをする

#### ダブルトップ

```
  　／＼／＼
　／　　　　＼
／　　　　　　＼
```

#### ヘッド・アンド・ショルダー

```
　　　　／＼
  ／＼／　　＼／＼
／　　　　　　　　＼
```

というような動きをしたときが損切りのポイント

また、チャネルラインのブレイクが発生したときなどが損切りのポイントとなる

定額や割合による損切りはしてはいけない
